Мінімум:

  1. Cпособи використання JS

    2. а) Внутрішнє підключенні в будь-якій частині файлу HTML (частіше перед закриваючим тегом </body<) з тегом <script< Недолік такого варіанту незручність при роботі з великим кодом та забрудненість HTML документа
    б) Зовнішнє підключення окремо створенного файлу з розширенням.js в файл HTML: <script src="./index.js" > </script> Найпоширеніший варіант. Також частіше розміщенний перед закриваючим тегом </body<
    в) Незалежнй скрипт (лише для зовнішніх скриптів). Завантажуються у фоновому режимі та запускаються по готовності. Асинхронне підключення - якщо потрібно завантажувати JavaScript код асинхронно, щоб не затримувати завантаження сторінки, можна використовувати атрибут async: <script async src="script.js"></script>
    г) Відкладений скрипт (призначений лише для зовнішніх скриптів). Скрипт завантажується “у фоновому режимі”, а потім запускається, коли DOM повністю побудовано. <script defer src="..."> </script>
    д) Динамічний скрипт
    let script = document.createElement('script');
    script.src = "...js";
    document.body.append(script);

  2. Дивись файл min.html

  3. Коментарі в JS бувають:

    4. а) Однострокові починаючи з //...
    б) Багатосрокові /*...*/
    - можна викоистовувати серед рядку кода;
    - можна як одностроковий;
    - можна як багатостроковий.
    в) Шебанг коментар #!
    Визначає шлях до JavaScript інтерпретатору, скрипт якого хочемо виконати.
    - дійсний тільки на початку скрипта;
    - дійсний до кінця першої строчки;

  4. Cтилі написання імен змінних бувають:

    5. Оголошуються змінні JavaScript за допомогою var, let або const.
    a) Ключове слово var використовується у всьому коді JavaScript з 1995 по 2015 рік.
    б) Ключові слова let та const було додано до JavaScript у 2015 році.
    в) Якщо хочете, щоб ваш код запускався в старішій версії веб-браузера, ви повинні використовувати var.

    Ідентифікатори:

    - Імена можуть містити літери, цифри, підкреслення та знаки долара.
    - Імена мають починатися з літери
    - Імена також можуть починатися з $ та _
    - Імена чутливі до регістру (y та Y - це різні змінні)
    - Зарезервовані слова (наприклад, ключові слова JavaScript) не можна використовувати як імена.
